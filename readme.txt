Adds a meta box to define a template for an EDD License key

== Description ==

Allows you to define the structure of an Easy Digital Downloads software license key. e.g CWP-9999-********-**99-** could generate CWP-2827-HS982KSI-A281-0K

A free plugin by [CalderaWP](https://CalderaWP.com).

== Installation ==

Install the plugin through the WordPress Plugins Installer or upload extracted folder to the plugins directory of your content directory
1. Activate the plugin through the 'Plugins' menu in WordPress
